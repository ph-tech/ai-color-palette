# 🎨 AI Color Palette Generator

An AI-powered tool that extracts the top 10 dominant colors from any uploaded image, built using Python and Streamlit.

## 🚀 Features

- 🔍 Extracts dominant colors from any image
- 🎨 Shows up to 10 most frequent colors with HEX values
- 💡 Simple UI using Streamlit
- 📁 Supports JPG, PNG, and other common image formats

---

## 🛠️ Tech Stack

- **Python**
- **Streamlit**
- **Pillow** (for image handling)
- **NumPy**
- **scikit-learn** (for KMeans clustering)

---

## 🧪 How It Works

- Upload any image through the Streamlit interface
- The app resizes the image for faster processing
- Uses **KMeans clustering** to find the most common colors
- Displays the result as a color palette with HEX codes
